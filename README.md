# Setup
### Requirements
Für die Ausführung der App mit Hilfe der `docker-compose.yml` wird Docker benötigt.

### Installation
Um die Applikation zu starten führen sie dies im Root-Verzeichnis des Projekts aus:
```
docker compose up -d
```
Danach ist die Applikation unter http://localhost:3000 erreichbar. Die API ist unter http://localhost:3000/items erreichbar.

## Hinweise
Es wurde `shadcn/ui` zusammen mit lucide-react als UI-Bibliothek verwendet.
